# Amazon Relay Urgent Site Alert

Auto-alerts for urgent work orders from priority sites.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/)
2. Click this link: [Install Script](https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_REPO_NAME/main/amazon-relay-urgent-alert.user.js)
3. Click "Install"

## Features
- Detects urgent sites: STL5, KCVG, SAV4, YVR2, RFD2, DFW7, LGA9, PSP1, CLEA, MCO*, CLE*
- Only triggers for unassigned work orders
- Copy-to-clipboard comment template
- Auto-updates daily
